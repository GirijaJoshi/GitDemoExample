5) git clone /path/to/repo (checkout repo)
6) git cone username@host:/path/tp/repo
7) git add <filename> to add specified file
8) git add * (to add all)
9) git commit -m "Commit message" (commit changes to head)
10) git commit -a (commit any files you have added with git)
11) git push origin master (save changes to master branch of your remote)
12) git status (list files status which have commited or changed)
13) git remote add origin <server> (if haven't connected your local repo to remote server to be able to push)
14) git remote -v (list all currently configured remote repos)
